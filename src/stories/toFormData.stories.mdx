# toFormData

A função `toFormData` recebe um objeto como argumento e gera a partir dele um novo `FormData`, 
usando as chaves como nomes e valores como valores

```js
const createUser = async data => {
  const payload = toFormData({
    name: data.name,
    picture: data.image // from <input type='file' />
  })

  await api.post('/users', formData)
}
```
