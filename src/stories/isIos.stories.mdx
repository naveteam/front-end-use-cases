# is IOS

Essa função tem como objetivo identificar se o usuário está acessando o sistema através de um dispositivo iOS (iPad/iPhone/iPod)

```js
export const isIOS = () => !!navigator.platform && /iPad|iPhone|iPod/.test(navigator.platform)
```
